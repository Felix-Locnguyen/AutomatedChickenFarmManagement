GHI CHÚ THIẾT KẾ:

1. Card Layout (Danh sách chuồng & Thiết bị gần đây):
   - border-left: 4px solid theo trạng thái
   - border-radius: 8px
   - padding: 10px 12px
   - min-height: 60-70px
   - Hover effect: translateY(-2px) + shadow
   - Tối ưu cho mobile

2. Trạng thái thiết bị gần đây:
   - Card hiển thị: Tên thiết bị + Tên chuồng + Trạng thái
   - Dấu chấm (●) màu theo trạng thái
   - Nhãn text: Hoạt động / Đang chờ / Ngắt kết nối

3. Card Layout thiết bị (trong chuồng):
   - Layout dọc: Tên → Pin/Kết nối → [Bật/Tắt] [Xóa]
   - Scrollable container giới hạn chiều cao
   - Tối ưu cho mobile (1 card/hàng)

4. Donut Chart:
   - Sơ đồ tròn rỗng với tổng ở giữa
   - Tương tác hover: hiện tooltip + đổi màu legend
   - Phù hợp hiển thị phân bố trạng thái

5. Logic Toggle:
   - Kiểm tra data-connection-status trước khi toggle
   - Toast notification với màu sắc tương ứng

6. Màu sắc chủ đạo:
   - Xanh lá: #22c55e (Hoạt động, Bật)
   - Vàng: #ffc107 (Chờ, Cảnh báo)
   - Đỏ: #dc3545 (Lỗi, Xóa)
   - Xám: #9ca3af (Tắt, Mặc định)
